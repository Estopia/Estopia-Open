# open.estopia.net

Base Next.js website for Estopia Engineering Ltd. to showcase open source software.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- ESLint

## Run Locally

```bash
npm install
npm run dev
```

Site runs at `http://localhost:3000`.

## Available Scripts

```bash
npm run dev    # Start dev server
npm run build  # Production build
npm run start  # Start production server
npm run lint   # Run ESLint
```

## What Is Included

- Branded homepage with:
	- Hero section
	- Featured project placeholders
	- Community contribution call-to-action
	- Footer
- Basic SEO metadata configured for `https://open.estopia.net`
