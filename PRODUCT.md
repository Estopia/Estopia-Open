# PRODUCT

## Product Name
open.estopia.net

## Register
brand

## Product Purpose
A public-facing brand surface for Estopia Engineering Ltd. that showcases open source software projects, communicates engineering values, and invites collaboration from developers and organisations.

## Users
- Developers evaluating Estopia open source tools
- Engineering leaders assessing tool reliability and maintenance signals
- Potential contributors looking for approachable projects and maintainer contact

## Brand Voice
Precise, practical, generous, credible, quietly ambitious.

## Strategic Principles
1. Show working value before marketing claims.
2. Prioritise clarity and trust over hype.
3. Make contribution pathways visible and low-friction.
4. Keep visual style distinctive without sacrificing readability.

## Anti-References
- Generic SaaS template heroes with repetitive metric cards
- Overly abstract copy that does not explain software outcomes
- Decorative UI that obscures key project information
